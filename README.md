## Installation

- Clone this repo:

```sh
git clone https://github.com/CodePapa360/The-Wild-Oasis.git
```

- Install dependencies:

```sh
npm install
```

- Build command:

```sh
npm run build
```

- Live server:

```sh
npm run dev
```
